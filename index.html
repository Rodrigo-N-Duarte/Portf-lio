Me chamo Rodrigo, tenho 19 anos de idade e moro em Betim - MG. Ainda não faço estágio na área, porém estou com a disposição e mente aberta para conseguir a primeira oportunidade no mercado da tecnologia.

CSS

Linguagem de estilização, trabalha junto com o HTML e da cores e visuais para as páginas em browsers.
Nível de conhecimento: Intermediário

JAVASCRIPT

o JavaScript pode sim ser considerada uma linguagem de programação legítima, foi criada para dar o funcionamento e "engenharia" dentro de uma página HTML, ou seja, no front-end. Porém, já está tão avançado que também é possível usá-la no back-end, como o NodeJS.
